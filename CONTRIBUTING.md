# About Contributing

1. Docusaurus 기반이기 때문에 대부분의 버그는 Docusaurus의 문제일 확률이 높습니다.  
그래도 문제가 있다고 생각하시면 Issue로 bug report를 작성해 주세요.  
Dive to Argo is using Docusaurus, so most bugs might be Docusaurus's problem.  
If you believe that you found a bug unrelated to this, please write an issue.

2. 번역 또는 내용에 있다면 Pull Request를 열어 주세요.  
내용에 문제가 있다면 정확한 Reference를 첨부해 주세요.  
If you find some translation or content issues, please open Pull Request.  
Attach correct reference if you report the content error.

3. 추가 문의는 haulrest@gmail.com으로 연락해 주세요.  
Please contact to email (haulrest@gmail.com) for additional request.
